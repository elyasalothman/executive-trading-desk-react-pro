# Executive Trading Desk React Pro

## المحتوى
- index.html
- style.css
- app.js

## التشغيل
1. فك الضغط عن الملف ZIP
2. افتح الملف `index.html` مباشرة في المتصفح
3. تأكد أن الإنترنت متصل حتى تظهر أدوات TradingView و Chart.js و React من الـ CDN

## المميزات المنفذة
- React بملفات منفصلة
- واجهة احترافية داكنة/فاتحة
- رسم بياني مباشر TradingView
- أجندة اقتصادية
- حاسبة مخاطر متقدمة
- سجل صفقات
- بحث وفلاتر
- إحصائيات شهرية
- رسم بياني للأداء الشهري
- تصدير CSV
- حفظ محلي كامل عبر localStorage
